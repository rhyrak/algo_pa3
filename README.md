## Usage

1) Run pa3.jar
   ```
   java -jar pa1.jar 1Mint.txt 50Mint.txt
   ```
2) Compile sources & run
   ```
   cd ./src
   javac *.java
   java Main
   ```
3) Create JAR file & run
   ```
   cd ./src
   javac *.java
   jar -cvfm pa3.jar META-INF/MANIFEST.MF *.class
   java -jar pa3.jar
   ```
